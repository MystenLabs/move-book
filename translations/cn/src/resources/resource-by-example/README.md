# Resource 示例

本节中，我们将学习如何定义、使用 Resource。最终我们将得到一份完整的智能合约，可以作为模板供日后使用。

我们将创建一个集合合约，它的功能包括：

- 创建一个集合
- 集合中添加、取出元素
- 回收集合
